Application class

Компонент приложения

Этот компонент создается первым и ему передается управление при запуске приложения в main.dart

Oтветственность компонента:

  • маршрутизация
  • инициализация компонентов бизнес-логики

Для реализации инициализации компонентов бизнес-логики при запуске и закрытии их при завершении приложения компонент реализует интерфейсы OnInit и OnDestroy.

Для вставки компонента в шаблон/html-документ используется элемент <app></app>. См. index.html
