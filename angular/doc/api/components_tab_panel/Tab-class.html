Tab class

Вкладка

Для вставки вкладки в шаблон используется элемент

Пример:

<tab-panel>
  <tab>Tab 1</tab>
  <tab>Tab 2</tab>
</tab-panel>

К компоненту Tab подмешивается EquatableMixin. Это позволяет избавиться от необходимости явно переопределять operator == и hashCode, что обычно делается при необходимости реализовать нестандартное сравнение объектов. В частности в нашем примере реализована возможность прямого сравнения вкладок, при котором вкладки считаются равными, если у них равны значения поля id.
