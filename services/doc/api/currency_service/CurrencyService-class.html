CurrencyService class

Интерфейс сервиса доступа к валютам

Объявляет базовые методы досупа к данным.

Все методы должны быть реализованя в конкретной реализации сервиса для выбранного способа хранения данных.

Простейшая реализация для хранения данных в оперативной памяти приведена в MemoryCurrencyService
