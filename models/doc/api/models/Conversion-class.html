Conversion class

Конверсия

Конверсия включает в себя валюту и соответствующую сумму. Конвертер валют может содержать однвременно несколько конверсий. При изменении валюты или суммы одной из них приводит к автоматическому пересчету значений остальных в соответствии с логикой, реализованной в компоненте бизнес-логики.
