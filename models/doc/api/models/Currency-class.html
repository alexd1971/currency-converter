Currency class

Валюта

К классу подмешивается EquatableMixin, который реализует функционал сравнения двух объектов на основании равенства значений определенных атрибутов экземпляров класса. Так в данном случае две валюты считаются равными, если у них равны коды валют. Список атрибутов, на основании которых определяется равенство объектов возвращает геттер props
