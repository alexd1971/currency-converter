Currency.from constructor

Currency.from(Currency currency)

Создает копию валюты

Передача объектов происходит не по значению, а по ссылке. Часто возникает задача вернуть из функции объект с новым состоянием, при этом состояние исходного объекта должно остаться прежним, чтобы не нарушить другую логику. В этом случае создается копия объекта и изменение состояние делается у копии.

Implementation

factory Currency.from(Currency currency) {
  return Currency(
      code: currency.code, name: currency.name, rate: currency.rate);
}
